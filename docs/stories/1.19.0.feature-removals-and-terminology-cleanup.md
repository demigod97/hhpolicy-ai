## Overview

Clean up the PolicyAi interface by removing deprecated features (Suggested Questions, Notes UI) that were identified as unnecessary through stakeholder feedback, update role terminology to be more user-friendly, and fix UI inconsistencies. This story represents the "Quick Wins" phase that must be completed before implementing new features in Epic 1.19.

---

## Business Requirements

**As a** Product Owner
**I want to** remove unused features and improve terminology clarity
**So that** the interface is cleaner, terminology is consistent, and users are not confused by deprecated features

**Value**:
- Reduced UI clutter improves user experience
- Clearer role terminology reduces confusion
- Fewer features to maintain and test
- Foundation for new template features
- Better alignment with actual user workflows

### 4. Update Filter Terminology

#### Create Role Label Mapping Utility:
```typescript
// src/lib/roleLabelMapping.ts (NEW FILE)

export const ROLE_DISPLAY_LABELS: Record<string, string> = {
  system_owner: 'System Owner',
  company_operator: 'Company Operator',
  administrator: 'General', // KEY CHANGE
  executive: 'Executive',
  board: 'Board',
};

export const ROLE_DESCRIPTIONS: Record<string, string> = {
  system_owner: 'Developer/backend access',
  company_operator: 'Document and user management',
  administrator: 'General staff access',
  executive: 'C-Level and VP access',
  board: 'Board member access',
};

export function getRoleDisplayLabel(role: string): string {
  return ROLE_DISPLAY_LABELS[role] || role;
}

export function getRoleDescription(role: string): string {
  return ROLE_DESCRIPTIONS[role] || '';
}
```

#### Update Filter Component:
```typescript
// src/components/dashboard/DocumentTableFilters.tsx

import { getRoleDisplayLabel } from '@/lib/roleLabelMapping';

  <label key={role} className="flex items-center gap-2 cursor-pointer">
    <span className="text-sm capitalize">{role.replace('_', ' ')}</span>
  </label>
))}

// AFTER:
{['administrator', 'executive', 'board', 'company_operator', 'system_owner'].map(role => (
  <label key={role} className="flex items-center gap-2 cursor-pointer">
    <span className="text-sm">{getRoleDisplayLabel(role)}</span>
  </label>
))}
```
