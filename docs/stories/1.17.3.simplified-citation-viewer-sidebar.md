# Story 1.17.3: Simplified Citation Viewer Sidebar

**Epic**: 1.17 Chat & Dashboard UX Enhancements
**Priority**: 🔴 **HIGH**
**Status**: ✅ **READY FOR REVIEW**
**Effort**: 0.3 days
**Created**: October 28, 2025

---

## Story

**As a** PolicyAi user
**I want to** see only the citation viewer when I click a citation, with "Save to Note" functionality
**So that** I can focus on the relevant document context without distraction, and the chat input remains accessible.

---

## Acceptance Criteria

1. **Remove Sources List from Sidebar**
   - Sources list no longer visible in chat sidebar
   - Users can view all documents in the Documents page instead
   - Sidebar only shows citation viewer when citation is clicked

2. **Citation Viewer at Top**
   - Citation viewer always positioned at top of sidebar
   - Scrollable within its own contained area
   - Does not push chat input off screen

3. **Add Save to Note Button**
   - "Save to Note" button displayed below citation content
   - Saves citation context to user notes
   - Uses existing NoteEditor/notes functionality

4. **Fixed Scrolling Behavior**
   - Citation viewer has independent scroll area
   - Main chat area scroll unaffected by sidebar
   - Chat input always visible at bottom
   - No need to scroll to access chat input

5. **Clean UI**
   - Sidebar shows empty state when no citation selected
   - Close button to dismiss citation viewer
   - Smooth transitions

## Dev Notes

### Current Implementation Issues

**File**: `src/components/chat/SourcesSidebar.tsx`

Current structure:
```
┌─────────────────────┐
│ Sources Header      │
│ (+ button, count)   │
├─────────────────────┤
│ Sources List        │  ← Remove this
│ (scrollable)        │
├─────────────────────┤
│ Separator           │
├─────────────────────┤
│ DocumentViewer      │  ← Keep this, move to top
│ (citation content)  │
└─────────────────────┘
```

Desired structure:
```
┌─────────────────────┐
│ Citation Header     │
│ (title, close btn)  │
├─────────────────────┤
│ Citation Content    │
│ (scrollable area)   │
│ - Document excerpt  │
│ - Highlighted lines │
├─────────────────────┤
│ Save to Note Button │
└─────────────────────┘
```

### Citation Viewer Component

**File**: `src/components/chat/DocumentViewer.tsx`

Current behavior:
- Shows document excerpt with highlighted lines
- Positioned at bottom of SourcesSidebar
- No save to note functionality

Changes needed:
- Add "Save to Note" button
- Ensure proper scroll containment
- Move to top of sidebar (structural change in SourcesSidebar)

### Scroll Issues

Problem: When citation viewer appears at bottom, user must scroll down to see it, which pushes chat input off screen.

Solution:
```tsx
<ScrollArea className="h-[calc(100vh-200px)]">
  <DocumentViewer ... />
</ScrollArea>
```

### Save to Note Integration

Use existing hooks:
- `useCreateNote` from `@/hooks/useNotes`
- Pass citation content as note content
- Default title: "Citation from [document name]"
