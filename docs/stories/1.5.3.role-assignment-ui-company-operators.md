# Story 1.5.3: Role Assignment UI for Company Operators

## Status: Approved

## Story

As a Company Operator,
I want a user management interface to assign roles to company users,
so that I can control access levels without requiring System Owner intervention.

## Context Source

- Source Document: Sprint Change Proposal - Epic 1.5
- Enhancement Type: User interface for role management
- Existing System Impact: New UI component for role assignment

## Acceptance Criteria

1. User Management Dashboard displays list of all company users with their current roles.
2. Company Operator can assign roles: Board Member, Administrator, Executive, Company Operator (but NOT System Owner).
3. Role assignment confirmation dialog shows clear description of role permissions before assignment.
4. Bulk role assignment capability for multiple users at once.
5. Audit log records all role changes with timestamp and operator information.
6. System Owners can assign any role including other System Owners.

## Dev Technical Guidance

### User Management Dashboard Components

**Main Components:**
- `UserManagementDashboard.tsx` - Main dashboard interface
- `UserTable.tsx` - Sortable user list with role display
- `RoleAssignmentDialog.tsx` - Role assignment modal
- `BulkActionBar.tsx` - Multi-select operations
- `UserDetailView.tsx` - Individual user details

**Role Assignment Logic:**
- Company Operators: Can assign all roles except System Owner
- System Owners: Can assign any role including other System Owners
- Role hierarchy validation before assignment
- Confirmation dialog with role description

### API Endpoints Required

**User Management:**
- `GET /api/admin/users` - List all users with roles
- `POST /api/admin/users/:id/role` - Assign role to user
- `POST /api/admin/users/bulk-role` - Bulk role assignment
- `GET /api/admin/users/:id/details` - User details and usage

**Role Validation:**
- Check if current user can assign target role
- Validate role hierarchy
- Log role changes for audit

### UI/UX Requirements

**Dashboard Layout:**
- User table with sortable columns
- Role filter dropdown
- Search functionality
- Bulk selection checkboxes
- Action buttons for role assignment

**Role Assignment Dialog:**
- Role selection dropdown
- Permission description display
- Confirmation step
- Success/error feedback

**Bulk Operations:**
- Multi-select user rows
- Bulk role assignment
- Bulk export functionality
- Progress indicators
