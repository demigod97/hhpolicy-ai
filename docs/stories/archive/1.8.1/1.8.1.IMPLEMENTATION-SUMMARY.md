## 🚀 Deployment Checklist

### Before Deployment

- [ ] Set OpenAI API key in Supabase secrets
- [ ] Configure `COPILOTKIT_LLM_MODEL` environment variable
- [ ] Update production `.env` with runtime URL
- [ ] Test Edge Function locally with `supabase functions serve`

### Deployment Steps

```bash
# 1. Set secrets
supabase secrets set OPENAI_API_KEY=sk-your-key-here
supabase secrets set COPILOTKIT_LLM_PROVIDER=openai
supabase secrets set COPILOTKIT_LLM_MODEL=gpt-4-turbo-preview

# 2. Deploy Edge Function
supabase functions deploy copilotkit-runtime

# 3. Verify deployment
curl -X POST https://your-project.supabase.co/functions/v1/copilotkit-runtime \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"messages":[{"role":"user","content":"test"}]}'
```
