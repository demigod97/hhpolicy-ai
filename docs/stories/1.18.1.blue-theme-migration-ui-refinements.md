## 📋 Story Overview

### Description
Transform the PolicyAi application from the current red (#EE4433) Human Habitat brand theme to a professional blue theme, while simultaneously implementing UI refinements to the Dashboard document table including:
1. Complete color theme migration (Red → Blue)
2. Removal of UserGreetingCard component from Dashboard
3. Enhanced DocumentTable styling with colored headers
4. Visual improvements for table presentation

### Business Value
- **Professional Appearance**: Blue theme provides more corporate, trustworthy appearance for policy/compliance software
- **Visual Hierarchy**: Colored table headers improve information scanning and readability
- **Streamlined UI**: Removing UserGreetingCard reduces clutter and improves mobile scrolling performance
- **Consistency**: Aligns with WorkflowMax-style enterprise UI patterns

### User Stories
```gherkin
As a PolicyAi user
I want a professional blue-themed interface with clear table headers
So that I can easily navigate policy documents in a corporate-appropriate UI

As an administrator
I want a streamlined dashboard without the greeting card
So that I can focus on document management tasks efficiently

As a mobile user
I want improved dashboard scrolling without collapsible sections
So that I can access documents quickly on my device
```
