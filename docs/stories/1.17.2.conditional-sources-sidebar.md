**Epic**: 1.17 Chat & Dashboard UX Enhancements

## Story

**As a** PolicyAi user
**I want to** see the sources sidebar only when I've started chatting and citations are present
**So that** I have more screen space when starting a new chat and the sources panel appears contextually when needed.

---

## Acceptance Criteria

1. **Initial State - No Sources Panel**
   - When chat session first loads, sources sidebar is NOT visible
   - Chat area takes full width (or with history sidebar only)
   - No empty sources panel cluttering the interface

2. **Conditional Display Logic**
   - Sources sidebar appears ONLY when ALL conditions are met:
     - User has sent at least one message in the session
     - AND at least one AI response contains citations
   - Sources sidebar remains visible once shown (doesn't hide after appearing)

3. **Citation Detection**
   - System detects citations in AI responses automatically
   - Citations parsed from markdown format `[^1]`, `[^2]`, etc.
   - Citation metadata extracted from message payload

4. **Responsive Behavior**
   - Desktop: Sources sidebar appears as third resizable panel on right
   - Mobile: Sources accessible via toggle button (existing pattern)
   - Smooth transition when sources panel appears

5. **Sources Sidebar Content (Unchanged)**
   - Shows all documents linked to chat session (existing functionality)
   - Citation click shows document viewer (existing functionality)
   - Collapsible sections work as before (existing functionality)

6. **State Persistence**
   - Sources sidebar visibility state maintained during session
   - Refreshing page respects chat history (shows sources if messages exist with citations)

### Citation Format

Citations appear in AI responses as:
- Markdown format: `[^1]`, `[^2]`, etc.
- Citation metadata in message object: `citations` array with source_id, page numbers
- Example: `{ sourceId: "abc-123", linesFrom: 10, linesTo: 15 }`

### Detection Logic Pseudocode

```typescript
// Check if user has sent messages
const hasUserMessages = messages.some(msg => msg.message.type === 'human');

// Check if any AI messages contain citations
const hasCitations = messages.some(msg =>
  msg.message.type === 'ai' && (
    msg.citations?.length > 0 ||
    /\[\^\d+\]/.test(msg.message.content)
  )
);

// Show sources sidebar if both conditions met
const showSourcesSidebar = hasUserMessages && hasCitations;
```
