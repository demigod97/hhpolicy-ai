#### Test 5: IV Uniqueness Simulation

```javascript
console.clear();
console.log('🧪 Test 5: IV Uniqueness');
console.log('=========================');

const ivs = new Set();
const iterations = 100;

// Generate 100 random IVs (simulating crypto.getRandomValues)
for (let i = 0; i < iterations; i++) {
  // Generate 12 random bytes (96 bits for AES-GCM)
  const iv = new Uint8Array(12);
  crypto.getRandomValues(iv);
  
  // Convert to hex string
  const ivHex = Array.from(iv)
    .map(b => b.toString(16).padStart(2, '0'))
    .join('');
  
  ivs.add(ivHex);
}

console.log(`Generated ${iterations} IVs`);
console.log(`Unique IVs: ${ivs.size}`);
console.assert(ivs.size === iterations, '✅ All IVs are unique');

console.log('Test 5 Complete\n');
```

// Check for exposed keys in various places
const sources = [
  document.body.innerHTML,
  document.body.innerText,
  JSON.stringify(window.localStorage),
  JSON.stringify(window.sessionStorage),
];

const patterns = [/sk-[a-zA-Z0-9]{20,}/, /AIza[a-zA-Z0-9]{20,}/];

sources.forEach((source, idx) => {

    }
  });
});

console.log('✅ Key exposure check complete');

## Troubleshooting

### Issue: Test component not visible

**Solution:** Ensure you're in development mode:
- Check `process.env.NODE_ENV === 'development'`
- Tab only shows in dev mode

### Issue: Supabase connection error

**Solution:**
```bash
supabase stop
supabase start
```

### Issue: Tests failing

**Check:**
1. Browser console for errors
2. Network tab for failed requests
3. Supabase logs
4. React DevTools for component state
